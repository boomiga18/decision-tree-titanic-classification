# Decision Trees for Classification — Titanic Tutorial

This is a beginner-friendly tutorial I created as part of my MSc Data Science coursework 
at the University of Hertfordshire. The goal was to teach Decision Trees in a simple and 
practical way using a real-world dataset.

## What is this about?

I used the Titanic dataset to build a Decision Tree classifier that predicts whether a 
passenger survived or not. I chose this topic because Decision Trees are one of the most 
intuitive machine learning algorithms — you can actually look at the tree and understand 
every decision it made.

## What is inside this repository?

- `notebook.ipynb` — the full Jupyter notebook with all the code, outputs, and plots
- `tutorial.pdf` — the written tutorial that explains each step clearly

## How to run the notebook

1. Open Google Colab at https://colab.research.google.com
2. Click File → Upload notebook → choose the .ipynb file
3. Run each cell one by one using Shift + Enter
4. No need to download any dataset — it loads automatically!

## What you will learn

- What a Decision Tree is and how it works
- How to clean and prepare real data
- How to train and evaluate a classifier using scikit-learn
- How to visualise a Decision Tree and read a Confusion Matrix

## Tools and libraries used

- Python 3
- pandas
- seaborn
- scikit-learn
